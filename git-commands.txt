Для ініціалізації репозиторія - пишемо git init
Поточний стан репозиторія - пишемо git status
//Щоб додати файл в staging area - пишемо git add <імя файлу>
Щоб зробити коміт - пишемо git commit -m <Опис коміта>
Щоб подивитися історію комітів - пишемо git log
Щоб перейти до конкретного коміту - пишемо git checkout <hash>
Щоб подивитися різницю між комітами - пишемо git dif <hash #1> <hash #2> <Назва файлу>
Щоб звязати віддалений репозиторій з локальним - пишемо один раз git remote add origin <адрес репозиторію>
Щоб подивитися список віддалених репозиторіїв - пишемо git remote --v
Щоб залити код на віддалений репозиторій - пишемо git push origin master
Щоб підтягнути код з віддаленого репозиторію в локальний - пишемо git pull origin master
Щоб обновляти існуючі файли в staging area - пишемо git add -u
