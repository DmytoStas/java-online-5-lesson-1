Для ініціалізації репозиторія - пишемо git init
Поточний стан репозиторія - пишемо git status
//Щоб додати файл в staging area - пишемо git add <імя файлу>
Щоб зробити коміт - пишемо git commit -m <Опис коміта>
Щоб подивитися історію комітів - пишемо git log
Щоб перейти до конкретного коміту - пишемо git checkout <hash>
